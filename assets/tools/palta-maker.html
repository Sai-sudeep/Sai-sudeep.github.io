🎶 Alankar / Palta Maker 🎶

📘 How to Use This Tool
+

1. Select or Enter a Scale

Choose a preset raga from the Presets dropdown or manually enter the Aaroh (ascending) and Avroh (descending) scale notes in the input fields.

2. Define a Pattern

Enter a numeric sequence (e.g., 1234) or a sequence of swaras (e.g., S G R) in the Pattern field. This defines the order of notes for your Alankar/Palta.

3. Use the Swara Keyboard

Click on the on-screen keyboard to insert special swaras, flats, sharps, and diacritics into any input field. Use Back, Clear, or Space buttons to edit the input conveniently.

4. Generate Alankar

Press the ✨ Generate Alankar ✨ button to produce the full ascending (Aaroh) and descending (Avroh) sequences based on your pattern and scale. The output appears in the large textarea.

5. Copy or Save Output

Click 📋 Copy Output to copy the generated Alankar to your clipboard for easy use in teaching, practice, or documentation.
